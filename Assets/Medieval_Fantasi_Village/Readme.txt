DemoController.cs
This File contains UI interaction for switching Buildings for Items Scene.

SoundController.cs
This file is used to play random sound with random volum pitch etc.

Functions:

void ChangeClip() - to change clip with random volume pitch and settings
public void PlayRandomSound() - start playing random sound 
public void PlaySound() - play none random sound
